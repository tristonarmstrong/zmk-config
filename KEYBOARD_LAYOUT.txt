  Bluetooth Pairing:
    BTCLR = Clear all pairings
    BT1-4 = Select/pair device slot 1-4
  
  Flashing:
    BOOT = Enter bootloader mode (hold RSE + X to flash firmware)


QUICK REFERENCE
──────────────────────────────────────────────────────────────────────────────
Layer Access:
  • LWR (hold) = Numbers, symbols, arrows
  • RSE (hold) = Function keys, media, Bluetooth

Common Shortcuts:
  • LWR + H/J/K/L = Arrow keys (Vim-style)
  • LWR + [ ] = Brackets
  • LWR + - = = Minus/Equal
  • RSE + VOL+/VOL- = Volume control
  • RSE + BTCLR = Clear Bluetooth pairings
